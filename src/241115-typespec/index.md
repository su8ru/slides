## 『オープンソースソフトウェア』

> オープンソースソフトウェア（英: Open Source Software、略称: OSS）とは、利用者の目的を問わずソースコードを使用、調査、再利用、修正、拡張、再配布が可能なソフトウェアの総称である。
>

## contribute の流れ

1. repository を fork する
2. commit する
3. pull request を投げる
4. reviewer とがんばる +α

## response で `@cookie` を指定されたときの挙動

パラメータではなく一般のプロパティにすることにした
（他の `@path` や `@query` と同様の挙動）
